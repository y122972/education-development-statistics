# 中国教育数据档案

将教育部 1998–2025 年《全国教育事业发展统计公报》整理成可查看最新年度、跨年趋势、可搜索数字原句、可回到官方原文核验的数据档案。

## 本地运行

```bash
pnpm install
pnpm dev
```

质量检查：

```bash
pnpm lint
pnpm build
```

## 数据结构

- `public/data/bulletins/<year>.md`：对应年度公报的完整正文、章节、注释和原图链接。
- `public/data/bulletins/<year>.json`：正文中的统计陈述，以及每个数值的语义标签、原始写法、数值和单位。
- `src/data/bulletins.ts`：年度索引、官方来源和前端数据类型。
- `src/data/educationData.ts`：从逐年公报中人工核对、统一单位后的跨年趋势数据。

当前 28 份结构化公报包含 1,248 条数字陈述、8,553 个数值。所有 JSON 均保留完整原句，可由前端“公报档案”搜索和核验。

## 数据原则

数据来源为[中华人民共和国教育部教育发展统计公报](https://www.moe.gov.cn/jyb_sjzl/sjzl_fztjgb/)。公报文案和统计口径会随年份变化，因此项目不使用固定关键词抓取脚本；每份公报均按正文语义人工提取，并保留原文作为审计依据。

- 原文未单列的同口径指标保留为空，不用推算值补齐。
- 亿人、万人、人数等在趋势数据中统一换算，原始精度仍以年度原文为准。
- 早期公报只公布“幼儿园园长和教师”合计时，不将其替代为学前教育专任教师；“普九人口覆盖率”等也不替代九年义务教育巩固率。
- 2021 年前后部分教育分类统计口径发生变化，跨年份比较时应同时查看对应公报原句。
- 图表用于观察趋势，正式引用请以教育部原始公报为准。

## 增加新年度

1. 在 `public/data/bulletins/` 增加该年度 `.md` 与 `.json`。
2. 在 `src/data/bulletins.ts` 增加年度索引和教育部原页。
3. 人工核对后，将可比指标追加到 `src/data/educationData.ts`。
4. 运行 `pnpm lint && pnpm build`，再检查趋势图、搜索和原文阅读视图。
